Spaceport
=========

All-in-one Vostok development infrastructure with integration tests inside.

Get Started
-----------

You need to have `docker` and `docker-compose` installed.
Then start services:

```
make
```

Useful controls
---------------

Prepare to launch
```
make prepare
```

Test integration (full cycle)
```
make test
```

Test integration (check only)
```
make check
```

Launch Vostok
```
make up
```

Clean spaceport
```
make clean
```
